# q1
Algebraic multiplicity is the number of times an eigenvalue appears 
as a root,  
while geometric multiplicity is the number of linearly
independent eigenvectors associated with that eigenvalue.

so geometric = 0 or 1  
and algebraic = 0 or nonzero (multiple)
